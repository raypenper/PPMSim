PPMSim: 1D Electromagnetic Wave Forward Modeling Script

PPMSim is an open-source Python package for 1D electromagnetic (EM) wave forward modeling, designed for applications such as radar sounding, layered media analysis, and geophysical interpretation. The model employs the propagation matrix method (PMM) to compute wave propagation and reflections through multi-layer media.
